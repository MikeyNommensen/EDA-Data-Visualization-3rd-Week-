# Feature Engineering - Week 3 Assignment

## Description

The project focuses on **Exploratory Data Analysis (EDA)** and **Feature Engineering** using Python.

The datasets are analyzed to understand the relationship between features and the target variable, as well as to prepare the data for machine learning models.
